# Stepper Motor Control
![Uploading kupitman.png…]()


Простой Python-проект для управления шаговым двигателем с GUI.  

> Этот репозиторий содержит один файл `ponos.py`, который можно запускать на компьютере для управления шаговым двигателем через Arduino (или совместимое устройство).

---

##  Возможности

- Подключение к COM-порту / USB для управления Arduino
- Анимация вращающейся стрелки в GUI (PyQt6)
- Настройка скорости вращения шагового двигателя
- Управление направлением вращения: по часовой стрелке (CW) и против часовой стрелки (CCW)

---

##  Требования

- Python 3.10+  
- PyQt6  
- pyserial
